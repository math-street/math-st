# Regev 2004

- [CITED] Oded Regev, “A Subexponential Time Algorithm for the Dihedral Hidden Subgroup Problem with Polynomial Space,” arXiv:quant-ph/0406151; Algorithmica 47 (2007), 29–39.
- [CITED] The paper modifies Kuperberg's approach to retain subexponential time while reducing space to polynomial.
- [PROVED] This establishes an algorithmic tradeoff and does not assign a hardware conversion to oracle queries.
- [PROVED] No numerical computation from this paper was used to support a repository result.
