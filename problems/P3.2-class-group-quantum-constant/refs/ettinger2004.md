# Ettinger–Høyer–Knill 2004

- [CITED] Mark Ettinger, Peter Høyer, and Emanuel Knill, “The Quantum Query Complexity of the Hidden Subgroup Problem Is Polynomial,” Information Processing Letters 91 (2004), 43–48, DOI 10.1016/j.ipl.2004.01.024; arXiv:quant-ph/0401083.
- [CITED] The paper identifies a hidden subgroup of an arbitrary finite group using a number of oracle calls polynomial in the logarithm of the group order, while allowing exponential total time.
- [PROVED] This upper bound separates query complexity from time complexity and blocks any universal subexponential query lower bound in the same oracle model.
