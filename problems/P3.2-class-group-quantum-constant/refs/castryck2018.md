# Castryck–Lange–Martindale–Panny–Renes 2018

- [CITED] Wouter Castryck, Tanja Lange, Chloe Martindale, Lorenz Panny, and Joost Renes, “CSIDH: An Efficient Post-Quantum Commutative Group Action,” ASIACRYPT 2018, 395–427, DOI 10.1007/978-3-030-03332-3_15; IACR ePrint 2018/383.
- [CITED] CSIDH instantiates a commutative ideal-class-group action with supersingular elliptic curves over a large prime field and uses it for noninteractive Diffie–Hellman-like key exchange.
- [PROVED] The toy experiment retains the prime-field trace-zero action structure but uses short-Weierstrass representatives and tiny discriminants, so it is not a parameter-level reproduction of CSIDH-512.
