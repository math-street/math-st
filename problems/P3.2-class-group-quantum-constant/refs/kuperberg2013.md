# Kuperberg 2013

- [CITED] Greg Kuperberg, “Another Subexponential-Time Quantum Algorithm for the Dihedral Hidden Subgroup Problem,” TQC 2013, LIPIcs 22, 20–34, DOI 10.4230/LIPIcs.TQC.2013.20.
- [CITED] The collimation sieve has rigorous `2^O(sqrt(log N))` quantum-time and classical-space bounds for a modified schedule, logarithmic quantum space, and heuristic time `soft-O(2^sqrt(2 log2 N))` for the optimized schedule.
- [CITED] Its parameters trade classical space and classical time against quantum time, and QRACM availability changes the preferred point.
- [PROVED] The fixed-batch pair simulator in this repository is not an implementation of the paper's full phase-vector collimation sieve; the state representation and recursion are different in the source code.
