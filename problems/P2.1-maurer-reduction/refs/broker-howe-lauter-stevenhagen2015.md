# Broker--Howe--Lauter--Stevenhagen 2015

**Source:** Reinier Broeker, Everett W. Howe, Kristin E. Lauter, and Peter
Stevenhagen, "Genus-2 Curves and Jacobians with a Given Number of Points,"
*LMS Journal of Computation and Mathematics* 18, 170--197, 2015.

**DOI:** <https://doi.org/10.1112/S1461157014000461>

**Preprint:** <https://arxiv.org/abs/1403.6911>

**Checked results:**

- [CITED] Theorem 1.1 and Corollary 4.8 imply exponential worst-case output
  size and running time for the genus-two CM construction from a prescribed
  Jacobian order.
- [CITED] The paper's heuristic algorithm that is polynomial away from a
  zero-density exceptional set prescribes the number of points on the
  genus-two curve itself.  It does not prescribe the order of the curve's
  Jacobian, which is the group needed by Maurer--Wolf.

**Use here:** [PROVED] A prescribed smooth order and a realizable Weil
polynomial cannot be converted into a polynomial-time uniform Maurer
auxiliary by invoking either result in this paper: the Jacobian-order CM
method has exponential worst cases, and the heuristic method targets the
wrong cardinality.
