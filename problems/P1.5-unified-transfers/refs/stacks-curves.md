# Stacks Project - Curves and function fields

- [CITED] The Stacks Project Authors, Section 53.2, "Curves and function
  fields," Lemma 53.2.2 (tag 0BXZ in section tag 0BXX).
- [CITED] The lemma states that rational maps from a normal curve to a proper
  variety are exactly morphisms, using the discrete-valuation extension at
  each missing point.
- [PROVED] A011 applies it with the smooth projective curve $E$ and a proper
  algebraic-group target.
