# Parent 1999

- [CITED] Pierre Parent, "Bornes effectives pour la torsion des courbes
  elliptiques sur les corps de nombres," *Journal fuer die reine und
  angewandte Mathematik* 506, 85--116, 1999,
  doi:10.1515/crll.1999.506.85.
- [CITED] Theorem 1.2 gives, for a rational point of prime-power order $p^a$
  over a degree-$d$ number field, the explicit upper bound
  $p^a\le65(3d-1)(2d)^6$ when $p\ge5$.
- [PROVED] A006 uses only the immediate consequence
  $d>(r/12480)^{1/7}$ for a rational point of prime order $r$.
- [PROVED] The consequence excludes standard explicit global torsion lifts
  from SG-01 because their degree and dense arithmetic cost are exponential in
  $\log r$.
