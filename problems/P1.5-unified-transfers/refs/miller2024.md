# Miller 1986/2024

- [CITED] Victor S. Miller, "Short Programs for Functions on Curves: A STOC
  Rejection," *12th International Conference on Fun with Algorithms*, LIPIcs
  291, 34:1--34:4, 2024, doi:10.4230/LIPIcs.FUN.2024.34; the paper republishes
  and contextualizes the 1986 STOC submission.
- [CITED] The work constructs functions on curves by straight-line programs
  and emphasizes that the program can be linear in the input size even when
  expanding the represented rational function is exponential.
- [PROVED] A009 uses this distinction to explain why A008's exponential
  geometric-degree requirement is compatible with polynomial-time Miller
  evaluation.
