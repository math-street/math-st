# Kopp--Lagarias 2024

- [CITED] Gene S. Kopp and Jeffrey C. Lagarias, "Ray Class Groups and Ray
  Class Fields for Orders of Number Fields," arXiv:2212.09177, revised
  November 30, 2024.
- [CITED] The paper develops ray class groups for arbitrary orders and proves
  exact sequences under simultaneous change of order and modulus. The ring
  class exact sequence used in A025--A027 is the trivial-modulus
  specialization of this framework.
- [CITED] The paper identifies ring/ray class fields of an order as explicit
  subfields of ray class fields of the maximal order. It does not construct a
  point-to-class character from a finite-field elliptic subgroup.

Primary source: <https://arxiv.org/abs/2212.09177>.
