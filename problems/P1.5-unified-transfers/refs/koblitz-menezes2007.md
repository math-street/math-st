# Koblitz--Menezes 2007

- [CITED] Neal Koblitz and Alfred Menezes, "Another Look at Generic Groups,"
  *Advances in Mathematics of Communications* 1(1), 13--28, 2007,
  doi:10.3934/amc.2007.1.13; IACR ePrint 2006/230.
- [CITED] The paper explains the generic-group model as excluding algorithms
  that exploit special features of a concrete representation and stresses
  that coordinate and implementation features can invalidate an attempted
  generic abstraction without invalidating Shoup's theorem.
- [PROVED] SG-14's boundary between random source handles and coordinate,
  lift, divisor, or pairing access is a problem-specific formalization of
  this established conceptual distinction. SG-14's counted transfer
  composition is repository work, but the generic-versus-concrete warning is
  not repository-original.

Primary sources: <https://doi.org/10.3934/amc.2007.1.13> and
