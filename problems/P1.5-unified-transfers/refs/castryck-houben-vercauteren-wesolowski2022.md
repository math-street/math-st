# Castryck--Houben--Vercauteren--Wesolowski 2022

- [CITED] Wouter Castryck, Marc Houben, Frederik Vercauteren, and Benjamin
  Wesolowski, "On the decisional Diffie--Hellman problem for class group
  actions on oriented elliptic curves," *Research in Number Theory* 8,
  article 99, 2022, doi:10.1007/s40993-022-00399-6.
- [CITED] Section 2.2 constructs the ideal action using
  $E[\mathfrak a]=\bigcap_{\alpha\in\mathfrak a}\ker\iota(\alpha)$ and states
  that the resulting oriented quotient depends only on $[\mathfrak a]$ and
  gives a free action.
- [PROVED] For P1.5 this fixes the direction of the standard CM mechanism:
  ideal classes act on oriented curves; it does not supply a character from an
  individual source point to the class group.
- [CITED] The paper uses Weil-pairing characters to learn information about an
  unknown acting ideal class, a separate decisional class-action problem.
