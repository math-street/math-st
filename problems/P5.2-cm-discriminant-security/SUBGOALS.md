# Sub-goals - P5.2

- [x] **SG-01a:** Construct one $D=-3$ and one $D=-4$ toy curve with a prime-order subgroup, verify each full order by exhaustive counting and BSGS point counting, and verify its explicit endomorphism equation on at least 32 seeded points.
- [x] **SG-02a:** Implement a shared Pollard-rho routine that canonicalizes under a supplied finite endomorphism orbit and returns a correct known logarithm on both special-CM families.
- [x] **SG-03a:** For subgroup orders spanning at least four bit sizes, run at least 200 seeded DLPs per family and baseline, then report collision-step ratios with 95% bootstrap confidence intervals.
- [x] **SG-04a:** Compare the $D=-3$ orbit of size six and the $D=-4$ orbit of size four separately, including exceptional shorter orbits.
- [x] **SG-05a:** Write a precise generic-group ceiling statement for finite, efficiently computable orbit actions and identify every assumption needed to turn it into a proof.
- [x] **SG-06a:** Decide whether an efficient CM endomorphism is a transfer under P1.5's operational definition.
- [x] **SG-07a:** Implement a $D=-7$ non-unit endomorphism, validate its characteristic equation, and measure the order of its eigenvalue on at least four prime-order subgroups.
- [x] **SG-07b:** Either give a polynomial-in-$\log r$ canonical representative for a growing non-unit orbit or prove a lower bound in an explicit evaluator model.
- [x] **SG-08a:** Determine whether an exponent-free or batched non-unit orbit representation can support a correct rho collision equation without yielding the exponent-returning canonicalizer used in the reduction.
- [ ] **SG-09a:** For the orientation-free collision constraint $(a+bs)/(c+ds)\in\langle\lambda\rangle$, determine whether multiple constraints admit a sub-$\sqrt r$ solver when the quotient index is two or three.
