# Satoh–Araki 1998

- [CITED] Satoh and Araki give a polynomial-time discrete-log algorithm for
  anomalous elliptic curves using Fermat-quotient/local-lifting machinery.
- [CITED] The algorithm therefore requires a local lift and arithmetic on
  lifted coordinates rather than only an opaque group interface.

Primary source: Takakazu Satoh and Kiyomichi Araki, Commentarii Mathematici
Universitatis Sancti Pauli 47(1), 1998, pp. 81–92,
DOI `10.14992/00009878`.
