# Maurer 2005

- [CITED] Maurer models an algorithm interacting with hidden state through
  declared operations and observable relations.
- [CITED] Equality, optionally augmented by an abstract order, recovers the
  generic-algorithm case in this framework.

Primary source: Ueli Maurer, Cryptography and Coding 2005, LNCS 3796,
pp. 1–12, `https://crypto.ethz.ch/publications/files/Maurer05.pdf`.
