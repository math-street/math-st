# Boneh–Boyen — BDHI IBE

Dan Boneh and Xavier Boyen, “Efficient Selective-ID Secure Identity-Based
Encryption Without Random Oracles,” EUROCRYPT 2004, LNCS 3027, 223–238.

Primary text:
<https://www.iacr.org/cryptodb/archive/2004/EUROCRYPT/1950/1950.pdf>

## Extracted statements

[CITED] Section 3.1 defines computational and decisional BDH, and Section 3.2
defines computational and decisional $q$-BDHI with target $e(g,g)^{1/x}$.

[CITED] The paper states that 1-BDHI is equivalent to BDH and that, at the time
of publication, equivalence between $q$-BDHI and BDH for $q>1$ was unknown.

[PROVED] P2.2 uses a type-3 lift of these symmetric experiments only when every
input and target group is written explicitly.
