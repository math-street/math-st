# Chase–Meiklejohn — Déjà Q

Melissa Chase and Sarah Meiklejohn, “Déjà Q: Using Dual Systems to Revisit
q-Type Assumptions,” EUROCRYPT 2014, LNCS 8441, 622–639.
DOI: 10.1007/978-3-642-55220-5_34.

Primary text: <https://eprint.iacr.org/2014/570.pdf>

## Extracted statements in repository notation

[CITED] The framework works in asymmetric composite-order bilinear groups and
reduces broad computational source-group polynomial assumptions to static
subgroup hiding plus parameter hiding.

[CITED] Example 4.10 instantiates the framework for the adaptive inverse-form
$q$-SDH relation by using $q+2$ hidden-subgroup shadow copies and an invertible
shifted-Vandermonde system.

[CITED] Theorem 4.8's hybrids give linear-in-$q$ loss for this instantiation,
up to negligible parameter-hiding and bad-event terms.
