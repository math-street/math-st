# Bowe 2017 — BLS12-381 construction note

**Reference.** [CITED] Sean Bowe, “BLS12-381: New zk-SNARK Elliptic Curve
Construction,” Electric Coin Company technical blog, March 11, 2017.
<https://electriccoin.co/blog/new-snark-curve/>

**Relevant source facts.** [CITED] The note publishes the BLS family parameter
\(u\), field values, and curve equations. It names low Hamming weight,
2-adicity, field size, and efficient-tower goals, but does not give an
exhaustive candidate order or first-passing rule.

**Audit use. [PROVED]** Once \(u\) and the family are fixed the core is derived, but
the number of screenable \(u\) values is not recoverable from this source.
