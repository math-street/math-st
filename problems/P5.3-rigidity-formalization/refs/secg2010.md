# SECG 2010 — SEC 2 version 2

**Reference.** [CITED] Standards for Efficient Cryptography Group,
*SEC 2: Recommended Elliptic Curve Domain Parameters*, version 2.0,
January 27, 2010. <https://www.secg.org/sec2-v2.pdf>

**Relevant source facts.** [CITED] Section 2.4.1 publishes the full
secp256k1 tuple \((p,a,b,G,n,h)\), including \(a=0\) and \(b=7\), and labels
it a Koblitz curve. The section contains no generation seed, finite candidate
menu, or stopping rule.

capacity is not identifiable.
